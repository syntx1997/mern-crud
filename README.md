## Installation

### Clone the Repository

```shell
git clone git@github.com:syntx1997/mern-crud.git
```

### Install Dependencies

```shell
npm i
```

```shell
cd client
```

```shell
npm i
```

```shell
cd ../
```

### Run frontend and backend concurrently

```shell
npm run dev
```
